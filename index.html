CSS-селектори

CSS-селектор – це частина CSS-правила, яка дозволяє вам вказати, до якого елемента (елементів) застосувати стиль.

У CSS дуже багато різних селекторів. Під час створення селектора ми можемо складати його з кількох селекторів для вибору необхідних елементів, до яких слід застосувати певні стилі.

  1. Селектори елементу - Як селектор може виступати будь-який елемент HTML, для якого визначаються правила форматування, такі як: колір, фон, розмір тощо.
  2. Селектори ідентифікатори - Ідентифікатор визначає унікальне ім'я елемента, яке використовується для зміни його стилю і звернення до нього через скрипти. Під час опису ідентифікатора спочатку вказується символ решітки (#), потім іде ім'я ідентифікатора. Повинен зустрічатися в коді документа тільки один раз.
  3. Селектор класу - Класи застосовують, коли необхідно визначити стиль для одного або декількох елементів веб-сторінки. Елементи при цьому можуть бути різними, але мати частково або повністю схожі елементи оформлення. У коді HTML ім'я класу визначається через атрибут class.
  4. Мультикласи - До будь-якого елемента одночасно можна додати кілька класів, перераховуючи їх в атрибуті class через пробіл. У цьому разі до елемента застосовується стиль, описаний у правилах для кожного класу. Оскільки під час додавання кількох класів вони можуть містити однакові стильові властивості, але з різними значеннями, то береться значення в класу, який описано в коді нижче.
  5. Універсальний селектор - Іноді потрібно встановити одночасно один стиль для всіх елементів веб-сторінки, наприклад, задати шрифт або накреслення тексту. У цьому випадку допоможе універсальний селектор, який відповідає будь-якому елементу веб-сторінки. * { Опис правил стилю }
  6. Вкладені селектори - Наприклад, встановити стиль для "b" тільки коли він розташовується всередині контейнера "p". Таким чином можна одночасно встановити стиль для окремого елемента, а також елемента, який знаходиться всередині іншого. E F { Опис правил стилю }
  7. Дочірні селектори - Дочірнім називається елемент, який безпосередньо розташовується всередині батьківського елемента. Який елемент виступає батьківським, а який його нащадком легко з'ясувати за допомогою дерева елементів - так називається структура відносин елементів документа між собою. E F { Опис правил стилю } Стиль застосовується до елемента F, тільки коли він є дочірнім елементом E.
  8. Сусідні селектори - Сусідні називаються елементи веб-сторінки, коли вони йдуть безпосередньо один за одним у коді документа. E + F { Опис правил стилю }
  9. Споріднені селектори - Споріднені селектори за своєю поведінкою схожі на сусідні селектори (запис виду E + F), але на відміну від них стильові правила застосовуються до всіх прилеглих елементів. Наприклад, для селектора h1~p стиль буде застосовуватися до всіх елементів "p", що розташовуються після заголовка "h1". При цьому "h1" і "p" повинні мати спільний батьківський елемент, тому якщо "p" вставити всередину "div", то стилі застосовуватися вже не будуть.
  10. [attr^='value'] - Встановлює стиль для елемента в тому випадку, якщо значення атрибута починається із зазначеного тексту.
  11. [атрибут$="значення"] - Встановлює стиль для елемента в тому випадку, якщо значення атрибута закінчується зазначеним текстом.
  12. [атрибут*="значення"] - Можливі варіанти, коли стиль слід застосувати до тега з певним атрибутом, коли частиною його значення є деякий текст. При цьому точно не відомо, в якому місці значення включено цей текст - на початку, в середині або наприкінці. У подібному випадку слід використовувати конструкцію *=. Вона визначає, що значення атрибута містить зазначений текст.
